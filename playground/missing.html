🔍 Missing Data Handling New in v0.0.1

tsb provides a full suite of missing-value utilities mirroring pandas — detection, filling, dropping, and interpolation.

1. Detection — isna / notna

isna(x) returns true for null, undefined, and NaN. Works on scalars, Series, and DataFrames.

2. Propagation Fill — ffill / bfill

Forward-fill (ffill) propagates the most recent non-null value forward. Backward-fill (bfill) fills from the next known value. Both accept an optional limit to restrict consecutive fills.

3. Fill with Value or Method — fillnaSeries

fillnaSeries extends the built-in Series.fillna(scalar) by adding ffill / bfill method support.

4. Drop Missing Values — dropnaSeries / dropnaDataFrame

dropnaDataFrame supports how, thresh, subset, and axis options — matching pandas' full API.

5. Interpolation — interpolate

Linear interpolation fills interior gaps using the two bounding non-null values. Leading and trailing nulls are left as-is (no extrapolation).

6. Real-world Example — Time Series Cleaning

Combining detection, filling, and interpolation to clean a sensor time series with sporadic missing readings.

API Reference

Function Description pandas equivalent
isna(x) / isnull(x) Detect missing values (scalar, Series, DataFrame) pd.isna()
notna(x) / notnull(x) Detect non-missing values pd.notna()
ffill(s, {limit?}) Forward-fill a Series Series.ffill()
bfill(s, {limit?}) Backward-fill a Series Series.bfill()
ffillDataFrame(df, {limit?}) Forward-fill all columns DataFrame.ffill()
bfillDataFrame(df, {limit?}) Backward-fill all columns DataFrame.bfill()
fillnaSeries(s, {value?, method?, limit?}) Fill with scalar or propagation method Series.fillna()
fillnaDataFrame(df, {value?, method?, limit?}) Fill DataFrame missing values DataFrame.fillna()
dropnaSeries(s) Remove missing values from Series Series.dropna()
dropnaDataFrame(df, {axis?, how?, thresh?, subset?}) Drop rows/columns with missing values DataFrame.dropna()
interpolate(s, {method?, limit?}) Linear interpolation Series.interpolate()
interpolateDataFrame(df, opts) Column-wise interpolation DataFrame.interpolate()
