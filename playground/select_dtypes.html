🔍 selectDtypes

selectDtypes(df, { include, exclude }) returns a new DataFrame containing only columns whose dtype satisfies the filter — just like DataFrame.select_dtypes().

Equivalent Python: df.select_dtypes(include=["number"])

Generic type aliases

AliasMatches dtypes
"number"int*, uint*, float*
"integer"int*, uint*
"signed"int*
"unsigned"uint*
"floating"float*
"bool"bool
"string"string
"datetime"datetime
"category"category
"object"object

1 · Include numeric columns only

const df = DataFrame.fromColumns({
  age:    [25, 30, 35],
  score:  [9.1, 8.2, 7.5],
  name:   ["Alice", "Bob", "Carol"],
  active: [true, false, true],
});

selectDtypes(df, { include: ["number"] }).columns.values;
// → ["age", "score"]

2 · Exclude numeric columns

selectDtypes(df, { exclude: ["number"] }).columns.values;
// → ["name", "active"]

3 · Include number, exclude floating → integers only

selectDtypes(df, { include: ["number"], exclude: ["floating"] }).columns.values;
// → ["age"]

4 · Exact dtype name

selectDtypes(df, { include: ["bool"] }).columns.values;
// → ["active"]

5 · Select datetime columns

const df2 = DataFrame.fromColumns({
  created: [new Date("2024-01-01"), new Date("2024-06-01")],
  value:   [100, 200],
});

selectDtypes(df2, { include: ["datetime"] }).columns.values;
// → ["created"]

6 · No options — returns DataFrame unchanged

selectDtypes(df).columns.values;
// → ["age", "score", "name", "active"]  (all columns)
