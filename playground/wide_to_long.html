↔️ wide_to_long

wideToLong reshapes a wide-format DataFrame into long format by gathering columns that share a common stub prefix, mirroring pandas.wide_to_long().

Equivalent Python: pandas.wide_to_long(df, stubnames, i, j)

1 · Basic wide-to-long

const df = { id: [1, 2], x1: [10, 20], x2: [11, 21] };
wideToLong(df, "x", "id", "time")
// id  time  x
//  1     1  10
//  2     1  20
//  1     2  11
//  2     2  21

2 · Multiple stubs

const df = { id: [1, 2], math1: [90, 80], math2: [95, 85], eng1: [70, 75], eng2: [72, 78] };
wideToLong(df, ["math", "eng"], "id", "year")

3 · Custom separator (sep: "_")

const df = { id: [1, 2], math_2020: [90, 80], math_2021: [95, 85], eng_2020: [70, 75], eng_2021: [72, 78] };
wideToLong(df, ["math", "eng"], "id", "year", { sep: "_" })

4 · String suffixes with custom suffix pattern

const df = { id: [1, 2], score_pre: [50, 60], score_post: [75, 80] };
wideToLong(df, "score", "id", "period", { sep: "_", suffix: "[a-z]+" })

5 · Multiple identifier columns

const df = { grp: ["a","b"], subgrp: [1, 2], v1: [10, 20], v2: [30, 40] };
wideToLong(df, "v", ["grp", "subgrp"], "t")

6 · Single-row input

const df = { name: ["Alice"], score1: [95], score2: [88], score3: [91] };
wideToLong(df, "score", "name", "exam")

7 · No stub columns → empty output

const df = { id: [1, 2], foo: [10, 20] };
wideToLong(df, "x", "id", "t")
// → 0 rows (no columns starting with "x")

8 · Numeric sort of suffixes

// Columns x10, x2, x1 → sorted as 1, 2, 10 (numeric)
const df = { id: [1], x10: [100], x2: [20], x1: [10] };
wideToLong(df, "x", "id", "t")
