melt — Wide to Long

Unpivot a DataFrame from wide format to long format — mirrors pandas.melt() / DataFrame.melt().

1 — Basic melt

Melt all value columns into a single variable / value pair.

import { DataFrame, melt } from "tsb";

const df = DataFrame.fromColumns({
  A: [1, 2, 3],
  B: [4, 5, 6],
  C: [7, 8, 9],
});

const long = melt(df);
console.log(long.toRecords());
// [{ variable: "A", value: 1 }, { variable: "A", value: 2 }, ...]

2 — Preserve identifier columns

Use id_vars to keep columns as identifiers that are repeated for each melted row.

import { DataFrame, melt } from "tsb";

const df = DataFrame.fromColumns({
  id:   ["alice", "bob", "carol"],
  jan:  [100, 200, 150],
  feb:  [110, 190, 160],
  mar:  [120, 210, 140],
});

const long = melt(df, { id_vars: "id", var_name: "month", value_name: "sales" });
console.log(long.toRecords());
▶ run

3 — Selective value columns

Use value_vars to specify which columns to unpivot.

import { DataFrame, melt } from "tsb";

const df = DataFrame.fromColumns({
  name:   ["Alice", "Bob"],
  score1: [85, 90],
  score2: [78, 92],
  grade:  ["B", "A"],
});

// Only melt score columns, keep name
const long = melt(df, {
  id_vars: "name",
  value_vars: ["score1", "score2"],
  var_name: "test",
  value_name: "score",
});
console.log(long.toRecords());
▶ run

4 — Multiple id columns

Pass an array to id_vars to use multiple identifier columns.

import { DataFrame, melt } from "tsb";

const df = DataFrame.fromColumns({
  year:  [2022, 2023],
  city:  ["NYC", "LA"],
  q1:    [100, 200],
  q2:    [110, 210],
});

const long = melt(df, {
  id_vars: ["year", "city"],
  var_name: "quarter",
  value_name: "revenue",
});
console.log(long.toRecords());
