🔍 idxmin / idxmax

Return the index label of the minimum or maximum value in a Series or DataFrame. Mirrors pandas.Series.idxmin() and pandas.DataFrame.idxmax().

Series: idxmin / idxmax

import { Series, idxmin, idxmax } from "tsb";

// Numeric series with custom string index
const s = new Series({ data: [3, 1, 4, 1, 5, 9, 2, 6], index: ["a","b","c","d","e","f","g","h"] });

idxmin(s);  // → "b"   (value 1 at position "b")
idxmax(s);  // → "f"   (value 9 at position "f")
idxmin → "b"   idxmax → "f"

Handling missing values (skipna)

const s2 = new Series({ data: [NaN, 2, 1], index: ["a", "b", "c"] });

idxmin(s2);                    // → "c"  (NaN skipped by default)
idxmin(s2, { skipna: false }); // → "a"  (NaN encountered first → return its label)
idxmin (skipna: true) → "c"    idxmin (skipna: false) → "a"

DataFrame: axis 0 — row label of extremum per column

import { DataFrame, idxminDataFrame, idxmaxDataFrame } from "tsb";

const df = DataFrame.fromColumns({
  price:    [120, 85, 200, 95],
  quantity: [5,   10,  2,  8],
  rating:   [4.5, 3.8, 4.9, 4.1],
});

idxminDataFrame(df);
// Series: { price: 1, quantity: 2, rating: 1 }
// (row 1 has min price, row 2 has min quantity, row 1 has min rating)

idxmaxDataFrame(df);
// Series: { price: 2, quantity: 1, rating: 2 }
columnidxminidxmax
price1 (85)2 (200)

DataFrame: axis 1 — column name of extremum per row

idxminDataFrame(df, { axis: 1 });
// For each row, which column has the smallest value?
// → Series: [0→"quantity", 1→"rating", 2→"quantity", 3→"rating"]
// (row 0: price=120, quantity=5, rating=4.5 → "quantity" is smallest)

idxmaxDataFrame(df, { axis: 1 });
// → Series: [0→"price", 1→"price", 2→"price", 3→"price"]
Note: When axis: 1, the result is indexed by the DataFrame's row index, and each value is the column name where the extremum occurs.
