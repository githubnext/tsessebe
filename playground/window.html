Window Functions New

tsb supports three families of window-based aggregation, mirroring the pandas rolling, expanding, and ewm APIs. All operate on Series (and column-wise on DataFrame).

Rolling Window

A rolling window aggregates over a fixed-size window of the most recent observations. Set minPeriods to allow partial windows; use center: true for symmetric windows.

Click "Run" to execute

Expanding Window

An expanding window grows with each observation: at position i it covers all values from 0 to i. This computes cumulative statistics.

Click "Run" to execute

Exponentially Weighted Moving (EWM)

EWM applies exponentially decreasing weights to past observations. Specify the decay using alpha (direct), span, com (center-of-mass), or halflife. Use adjust: false for the recursive formula.

Rolling on DataFrame

All rolling aggregation functions also work on DataFrame, computing each column independently.
