🗺️ map / transform

seriesMap maps each element of a Series through a function, Map, plain-object dict, or another Series — mirroring pandas.Series.map. dataFrameTransform applies a function to each column (or row) and returns a same-shaped DataFrame — mirroring pandas.DataFrame.transform.

seriesMap

Map Series values through a function, Map, dict, or lookup Series.

Function
Dict
Series lookup
import { Series, seriesMap } from "tsb";

const s = new Series({ data: [1, 2, null, 4] });
seriesMap(s, (v) => (v as number) * 10).values;
// [10, 20, null, 40]   ← null passed to fn by default

// na_action="ignore" skips missing values
seriesMap(s, (v) => (v as number) * 10, { naAction: "ignore" }).values;
// [10, 20, null, 40]   ← null preserved unchanged
import { Series, seriesMap } from "tsb";

const s = new Series({ data: ["a", "b", "c", "d"] });
seriesMap(s, { a: 1, b: 2, c: 3 }).values;
// [1, 2, 3, undefined]   ← "d" not in dict → undefined
import { Series, seriesMap } from "tsb";

const lookup = new Series({ data: ["alpha","beta","gamma"], index: [1,2,3] });
const s = new Series({ data: [2, 1, 3, 99] });
seriesMap(s, lookup).values;
// ["beta", "alpha", "gamma", undefined]

dataFrameTransform

Apply a function to each column (axis=0) or row (axis=1) of a DataFrame, returning a same-shaped result.

Column-wise (axis=0)
Row-wise (axis=1)
Per-column dict
import { DataFrame, dataFrameTransform } from "tsb";

const df = DataFrame.fromColumns({ a: [1,2,3], b: [10,20,30] });
// Subtract column mean from each value (z-score numerator)
dataFrameTransform(df, (col) => {
  const mean = col.values.reduce((s,v) => s + (v as number), 0) / col.values.length;
  return col.values.map((v) => (v as number) - mean);
});
import { DataFrame, dataFrameTransform } from "tsb";

const df = DataFrame.fromColumns({ a: [1,2,3], b: [4,5,6] });
// Negate every value in each row
dataFrameTransform(df, (row) => row.values.map((v) => -(v as number)), { axis: 1 });
import { DataFrame, dataFrameTransform } from "tsb";

const df = DataFrame.fromColumns({ a: [1,2,3], b: [4,5,6], c: [7,8,9] });
// Only transform columns a and b; c passes through unchanged
dataFrameTransform(df, {
  a: (col) => col.values.map((v) => (v as number) * 100),
  b: (col) => col.values.map((v) => -(v as number)),
});
