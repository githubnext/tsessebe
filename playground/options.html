📦 pd.options system

The options system mirrors pandas.get_option / pandas.set_option. It manages display, mode, and compute settings with full validation support.

Cell 1 — getOption / setOption / resetOption

import { getOption, setOption, resetOption } from "tsb";

getOption("display.max_rows");          // 60
setOption("display.max_rows", 20);
getOption("display.max_rows");          // 20
resetOption("display.max_rows");
getOption("display.max_rows");          // 60 (restored)

Cell 3 — optionContext (scoped override)

import { optionContext, getOption } from "tsb";

const ctx = optionContext("display.max_rows", 5, "display.precision", 2);
ctx.enter();
// inside context:
getOption("display.max_rows");   // 5
getOption("display.precision");  // 2
ctx.exit();
// restored:
getOption("display.max_rows");   // 60
