🔀 explode

explodeSeries and explodeDataFrame transform list-valued cells into multiple rows — one row per element. Mirrors pandas.DataFrame.explode.

Demo 1 Basic Series explode

import { Series, explodeSeries } from "tsb";

const s = new Series({ data: [[1, 2, 3], [4, 5], [6]], name: "nums" });
const result = explodeSeries(s);
// Values: [1, 2, 3, 4, 5, 6]
// Index:  [0, 0, 1, 1, 2]  ← original labels repeated

Demo 4 DataFrame explode — single column

import { DataFrame, explodeDataFrame } from "tsb";

const df = DataFrame.fromColumns({
  id:   [1, 2, 3],
  tags: [["ts","js"], ["python"], ["go","rust","c++"]],
});
explodeDataFrame(df, "tags");
// id  tags
//  1  ts
//  1  js
//  2  python
//  3  go
