📈 interpolate

interpolateSeries / interpolateDataFrame fill missing (null / NaN) values using one of several interpolation methods, just like Series.interpolate().

Equivalent Python: series.interpolate(method="linear")

1 · Linear interpolation (default)

const s = new Series({ data: [0, null, null, null, 4] });
interpolateSeries(s).values;
// → [0, 1, 2, 3, 4]

2 · Edge missing values are not extrapolated

const s = new Series({ data: [null, null, 2, null, 4, null] });
interpolateSeries(s).values;
// → [null, null, 2, 3, 4, null]  (leading/trailing unchanged)

3 · Pad / forward-fill

const s = new Series({ data: [1, null, null, 4, null] });
interpolateSeries(s, { method: "pad" }).values;
// → [1, 1, 1, 4, 4]

4 · Backfill / backward-fill

const s = new Series({ data: [null, null, 3, null, 5] });
interpolateSeries(s, { method: "bfill" }).values;
// → [3, 3, 3, 5, 5]

5 · Nearest-neighbour

const s = new Series({ data: [1, null, null, null, 5] });
interpolateSeries(s, { method: "nearest" }).values;
// → [1, 1, 1, 5, 5]  (pos 1,2 → left; pos 3 → right)

6 · Limit consecutive fills

const s = new Series({ data: [0, null, null, null, null, 5] });
interpolateSeries(s, { limit: 1 }).values;
// → [0, 1, null, null, 4.8, 5]  (at most 1 fill from each neighbour)

7 · DataFrame — interpolate each column (axis=0)

const df = DataFrame.fromColumns({
  temperature: [15, null, null, 21],
  humidity:    [60, 62, null, 66],
});
interpolateDataFrame(df).toString();

8 · DataFrame — interpolate across rows (axis=1)

const df = DataFrame.fromColumns({
  mon: [10, 0], tue: [null, null], wed: [20, 8],
});
interpolateDataFrame(df, { axis: 1 }).toString();
