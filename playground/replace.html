🔄 replace

replace substitutes one or more values in a Series or DataFrame, mirroring pandas.Series.replace / pandas.DataFrame.replace.

API

// Scalar pair
replaceSeries(series, oldValue, newValue)

// List pair — parallel arrays
replaceSeries(series, [old1, old2], [new1, new2])

// Dict form — { "encodedOld": newValue }
replaceSeries(series, { "1": 99, "null": 0 })

// DataFrame — global
replaceDataFrame(df, 1, 99)

// DataFrame — per-column nested dict
replaceDataFrame(df, { colA: { "1": 99 }, colB: { "2": 0 } })

Key facts

  • Missing values (null, undefined, NaN) can be replaced — use the string keys "null", "undefined", or "NaN" in the dict form.
  • The original Series/DataFrame is never mutated — a new object is returned.
  • Index and column names are always preserved.
  • Unmatched values pass through unchanged.
  • Per-column replacement (DataFrame) uses a nested dict: { colName: { old: new } }.
