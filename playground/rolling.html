Rolling Windows

Sliding-window aggregations — mirrors pandas.Series.rolling() and pandas.DataFrame.rolling().

1 — Basic rolling mean

Call series.rolling(window) to get a Rolling object, then call .mean().

import { Series } from "tsb";

const prices = new Series({
  data: [100, 102, 98, 105, 110, 108, 112],
  name: "price",
});

// 3-day rolling mean — first two positions are null (not enough data)
const ma3 = prices.rolling(3).mean();
console.log(ma3.toArray());
▶ run

2 — minPeriods: allow partial windows

By default minPeriods = window. Set it lower to get results for the initial positions too.

import { Series } from "tsb";

const s = new Series({ data: [10, 20, 30, 40, 50] });

// window=3 but minPeriods=1 → start computing from index 0
const result = s.rolling(3, { minPeriods: 1 }).mean();
console.log(result.toArray()); // [10, 15, 20, 30, 40]
▶ run

3 — Rolling sum, std, var, min, max

import { Series } from "tsb";

const s = new Series({ data: [2, 4, 4, 4, 5, 5, 7, 9] });
const w = s.rolling(3);

console.log("sum :", w.sum().toArray());
console.log("std :", w.std().toArray().map(v =>
  v === null ? null : Number(v.toFixed(4))));
console.log("min :", w.min().toArray());
console.log("max :", w.max().toArray());
▶ run

4 — Rolling count (handles nulls)

count() counts valid (non-null / non-NaN) observations in each window. It ignores minPeriods.

import { Series } from "tsb";

const s = new Series({ data: [1, null, 3, null, 5] });

// count valid observations per window
console.log(s.rolling(2).count().toArray()); // [1, 1, 1, 1, 1]
console.log(s.rolling(3).count().toArray()); // [1, 1, 2, 1, 2]

// mean() with minPeriods=1 skips null-only windows
console.log(s.rolling(2, { minPeriods: 1 }).mean().toArray());
▶ run

5 — Rolling median

Useful for robust estimation — less sensitive to outliers than the mean.

import { Series } from "tsb";

const s = new Series({ data: [1, 3, 5, 1, 100, 2, 4] });

const meanW = s.rolling(3, { minPeriods: 1 }).mean().toArray();
const medW  = s.rolling(3, { minPeriods: 1 }).median().toArray();

console.log("mean  :", meanW.map(v => v === null ? null : +v.toFixed(2)));
console.log("median:", medW);
▶ run

6 — Custom aggregation with apply()

Pass any function (values: readonly number[]) => number to apply().

import { Series } from "tsb";

const s = new Series({ data: [1, 2, 3, 4, 5] });

// Rolling range (max - min)
const range = s.rolling(3).apply(vals =>
  Math.max(...vals) - Math.min(...vals)
);
console.log("range:", range.toArray()); // [null, null, 2, 2, 2]

// Rolling product
const prod = s.rolling(2).apply(vals => vals.reduce((a, b) => a * b, 1));
console.log("prod :", prod.toArray()); // [null, 2, 6, 12, 20]
▶ run

7 — Centered window

Set center: true to centre the window label, giving a symmetric look-ahead/look-behind view.

import { Series } from "tsb";

const s = new Series({ data: [1, 2, 3, 4, 5] });

// trailing (default)
const trailing = s.rolling(3).mean().toArray();
// centered
const centered = s.rolling(3, { center: true, minPeriods: 1 }).mean().toArray();

console.log("trailing:", trailing);  // [null, null, 2, 3, 4]
console.log("centered:", centered); // [1.5, 2, 3, 4, 4.5]
▶ run

8 — DataFrame.rolling()

Aggregations are applied column-by-column, returning a new DataFrame with the same shape.

import { DataFrame } from "tsb";

const df = DataFrame.fromColumns({
  open:  [100, 101, 103, 102, 105],
  close: [101, 103, 102, 105, 107],
});

const rolled = df.rolling(3).mean();
console.log(rolled.col("open").toArray());
console.log(rolled.col("close").toArray());
▶ run
💡 Tip: chain rolling methods — e.g. df.rolling(5).std() gives a DataFrame of 5-day rolling standard deviations across all numeric columns. Compare with .mean() to spot volatility spikes.
