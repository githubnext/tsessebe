🔢 abs & round

absSeries / absDataFrame return element-wise absolute values for numeric data. roundSeries / roundDataFrame round each numeric element to a given number of decimal places. Both mirror the pandas .abs() and .round() methods. Non-numeric values (null, strings, booleans) pass through unchanged. 

import { absSeries, roundSeries, absDataFrame, roundDataFrame } from "tsb";
import { Series, DataFrame } from "tsb";

// Series.abs()
const s = new Series([-3.5, 1.2, null, -0.8]);
absSeries(s).values;         // [3.5, 1.2, null, 0.8]

// Series.round(decimals)
roundSeries(s, 1).values;    // [-3.5, 1.2, null, -0.8]
roundSeries(s, 0).values;    // [-4, 1, null, -1]

// DataFrame.abs()
const df = DataFrame.fromColumns({ a: [-1, 2, -3], b: [3.14, -2.71, 0] });
absDataFrame(df).toRecords();
// [{ a: 1, b: 3.14 }, { a: 2, b: 2.71 }, { a: 3, b: 0 }]

// DataFrame.round() — scalar or per-column dict
roundDataFrame(df, 1).toRecords();
roundDataFrame(df, { a: 0, b: 2 }).toRecords();

Behaviour notes

  • null / undefined / NaN pass through unchanged for both abs and round.
  • Strings and booleans also pass through unchanged (only numeric values are transformed).
  • Negative decimals rounds to the nearest 10, 100, etc. — e.g. roundSeries(s, -1) rounds to tens.
  • For roundDataFrame with a per-column dict, columns not in the dict are left unchanged.
