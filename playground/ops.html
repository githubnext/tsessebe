🔢 Alignment & Arithmetic

Index-aware arithmetic that aligns on labels before operating — just like pandas.

1. Why alignment matters

In a labeled Series, adding by position silently produces wrong answers when the indexes don't match. tsb aligns on labels first, just like pandas. 


      

    


    
    

      
2. Mismatched indexes produce NaN

      

        When a label is present in one Series but not the other, the result is NaN
        (the pandas convention for "missing after alignment").
      

      

        
        
        

      

    


    
    

      
3. Use fillValue to avoid NaN

      

        Pass a fillValue as the second argument to substitute for missing values
        before the operation is applied.
      

      

        
        
        

      

    


    
    

      
4. alignSeries() — inspect alignment

      

        alignSeries(a, b, join?) returns { left, right, index } so
        you can inspect exactly how two Series align before operating.
      

      

        
        
        

      

    


    
    

      
5. alignedBinaryOp() — custom binary ops

      

        Apply any binary function to two aligned Series.  Useful for operations beyond
        the built-in add / sub / mul / div.
      

      

        
        
        

      

    


    
    

      
6. DataFrame arithmetic

      

        DataFrames support the same arithmetic API: both scalar and DataFrame operands.
        With a DataFrame operand, both rows and columns are aligned.
      

      

        
        
        

      

    


    
    

      
7. DataFrame alignment with mismatched columns

      

        When two DataFrames have different columns, the result includes all columns
        (outer join).  Missing cells become null (or use fillValue).
