🔢 factorize

factorize encodes an array of values as integer codes, just like pandas.factorize(). Each unique value gets a 0-based code; missing values get a sentinel (default -1).

Equivalent Python: pandas.factorize(values) / Series.factorize()

1 · Basic encoding (first-seen order)

const { codes, uniques } = factorize(["cat", "dog", "cat", null, "dog"]);
// codes:   [0, 1, 0, -1, 1]
// uniques: ["cat", "dog"]

2 · Sorted encoding (sort: true)

const { codes, uniques } = factorize(["banana", "apple", "cherry", "apple"], { sort: true });
// codes:   [1, 0, 2, 0]
// uniques: ["apple", "banana", "cherry"]

3 · Custom missing sentinel (naValue)

const { codes } = factorize(["a", null, undefined, "b", "a"], { naValue: -99 });
// codes: [0, -99, -99, 1, 0]

4 · Numeric values

const { codes, uniques } = factorize([3, 1, 4, 1, 5, 9, 2, 6]);
// codes:   [0, 1, 2, 1, 3, 4, 5, 6]
// uniques: [3, 1, 4, 5, 9, 2, 6]

5 · All-missing input

const { codes, uniques } = factorize([null, null, undefined]);
// codes:   [-1, -1, -1]
// uniques: []

6 · NaN is treated as missing

const { codes, uniques } = factorize([1, NaN, 2, NaN, 1]);
// codes:   [0, -1, 1, -1, 0]
// uniques: [1, 2]

7 · Boolean values

const { codes, uniques } = factorize([true, false, true, null, false]);
// codes:   [0, 1, 0, -1, 1]
// uniques: [true, false]

8 · Treat null as a regular value (naValue: null)

const { codes, uniques } = factorize(["x", null, "y", null], { naValue: null });
// null gets a real code alongside "x" and "y"
