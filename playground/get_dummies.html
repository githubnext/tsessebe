getDummies one-hot encoding

Convert categorical variables into binary indicator columns — mirrors pandas.get_dummies and pandas.from_dummies.

1. Basic Series → dummy DataFrame

Input Series
const s = new Series({
  data: ["cat", "dog", "cat", "fish"],
  name: "animal"
});
