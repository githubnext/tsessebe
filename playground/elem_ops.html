element-wise operations

Scalar transforms applied independently to each element — mirrors pandas.Series.clip(), .abs(), and .round().

1 — clip: bound values to a range

clip(series, { lower, upper }) replaces any value below lower with lower, and any value above upper with upper. Pass only one bound to clip from one side only. Mirrors pandas.Series.clip(). 

import { Series, clip } from "tsb";

const s = new Series({ data: [-3, 0, 1, 5, 10] });
const clipped = clip(s, { lower: 0, upper: 6 });
console.log([...clipped.values]); // [0, 0, 1, 5, 6]

// One-sided: only lower bound
const pos = clip(s, { lower: 0 });
console.log([...pos.values]); // [0, 0, 1, 5, 10]

2 — clip on a DataFrame

dataFrameClip(df, { lower, upper }) applies the same clipping to every numeric column. Mirrors pandas.DataFrame.clip(). 

import { DataFrame, dataFrameClip } from "tsb";

const df = DataFrame.fromColumns({
  temperature: [-10, 5, 20, 45],
  humidity:    [  0, 30, 80, 110],
});
const safe = dataFrameClip(df, { lower: 0, upper: 100 });
console.log([...safe.col("temperature").values]); // [0, 5, 20, 45]
console.log([...safe.col("humidity").values]);     // [0, 30, 80, 100]

3 — seriesAbs: absolute value

seriesAbs(series) returns a new Series where every element is replaced by its absolute value. Mirrors pandas.Series.abs(). 

import { Series, seriesAbs } from "tsb";

const returns = new Series({ data: [-0.05, 0.12, -0.08, 0.03] });
const magnitude = seriesAbs(returns);
console.log([...magnitude.values]); // [0.05, 0.12, 0.08, 0.03]

4 — dataFrameAbs: absolute values for all columns

dataFrameAbs(df) applies abs() column-by-column. Mirrors pandas.DataFrame.abs(). 

import { DataFrame, dataFrameAbs } from "tsb";

const df = DataFrame.fromColumns({
  x: [-1, 2, -3],
  y: [ 4, -5, 6],
});
const abs = dataFrameAbs(df);
console.log([...abs.col("x").values]); // [1, 2, 3]
console.log([...abs.col("y").values]); // [4, 5, 6]

5 — seriesRound: round to N decimal places

seriesRound(series, { decimals }) rounds each value to the given number of decimal places (default 0). Negative decimals rounds to the left of the decimal point (e.g. -1 rounds to the nearest 10). Mirrors pandas.Series.round(). 

import { Series, seriesRound } from "tsb";

const prices = new Series({ data: [1.2349, 9.9999, 3.14159] });
const r2 = seriesRound(prices, { decimals: 2 });
console.log([...r2.values]); // [1.23, 10.0, 3.14]

// Round to nearest 10
const big = new Series({ data: [14, 25, 36] });
console.log([...seriesRound(big, { decimals: -1 }).values]); // [10, 30, 40]

6 — dataFrameRound: round all columns

dataFrameRound(df, { decimals }) rounds every numeric column of a DataFrame. Mirrors pandas.DataFrame.round(). 

import { DataFrame, dataFrameRound } from "tsb";

const df = DataFrame.fromColumns({
  lat: [51.5074,  48.8566, 40.7128],
  lon: [-0.1278,   2.3522, -74.006],
});
const r2 = dataFrameRound(df, { decimals: 2 });
console.log([...r2.col("lat").values]); // [51.51, 48.86, 40.71]
console.log([...r2.col("lon").values]); // [-0.13, 2.35, -74.01]

7 — missing values pass through

All three operations propagate null and NaN unchanged — consistent with pandas' behaviour. 

import { Series, clip, seriesAbs, seriesRound } from "tsb";

const s = new Series({ data: [null, -5, NaN, 7] });

console.log([...clip(s, { lower: 0, upper: 5 }).values]);
// [null, 0, NaN, 5]

console.log([...seriesAbs(s).values]);
// [null, 5, NaN, 7]

console.log([...seriesRound(s, { decimals: 0 }).values]);
// [null, -5, NaN, 7]
