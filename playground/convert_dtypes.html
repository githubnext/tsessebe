convert_dtypes dtype inference

convert_dtypes infers the best dtype for each column of a Series or DataFrame — mirroring pandas.DataFrame.convert_dtypes(). Integer-valued floats become int64, string columns become string, and boolean columns become bool. Null values are always preserved.

API

import { convertDtypes, dataFrameConvertDtypes } from "tsb";

// Series
const s = new Series({ data: [1.0, 2.0, null, 3.0] });
const s2 = convertDtypes(s);
// s2.dtype.name === "int64"  ← whole-number floats promoted to int

// DataFrame — each column converted independently
const df = DataFrame.fromColumns({
  a: [1.0, 2.0, 3.0],       // whole-number floats → int64
  b: ["x", "y", "z"],        // strings             → string
  c: [true, false, true],    // booleans             → bool
  d: [1.5, 2.5, 3.5],        // fractional floats   → float64
});
const df2 = dataFrameConvertDtypes(df);

// Options (all default true):
dataFrameConvertDtypes(df, {
  convertInteger: true,   // promote whole-number cols → int64
  convertBoolean: true,   // promote boolean cols → bool
  convertFloating: true,  // promote float cols → float64
  convertString:  true,   // promote string cols → string
});

Idempotence

Calling convert_dtypes twice on the same Series or DataFrame always gives the same result as calling it once — the inferred dtype is stable once applied.
