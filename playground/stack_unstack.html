stack / unstack

Pivot column labels into the row index and back — mirrors pandas.DataFrame.stack() / pandas.Series.unstack().

1 — Basic stack

stack(df) rotates column labels into the row index, producing a flat Series. Index labels take the form "rowLabel|colName".

import { DataFrame, stack } from "tsb";

const df = DataFrame.fromColumns(
  { A: [1, 2, 3], B: [4, 5, 6] },
  { index: ["x", "y", "z"] },
);

const s = stack(df, { dropna: false });
console.log([...s.index.values]); // ["x|A","x|B","y|A","y|B","z|A","z|B"]
console.log([...s.values]);        // [1, 4, 2, 5, 3, 6]

2 — stack drops null by default

Like pandas, stack() silently omits cells whose value is null or NaN. Pass { dropna: false } to keep them.

import { DataFrame, stack } from "tsb";

const df = DataFrame.fromColumns(
  { score: [10, null, 30], bonus: [null, 2, 3] },
  { index: ["a", "b", "c"] },
);

// default: dropna=true
const s = stack(df);
console.log([...s.index.values]); // ["a|score","b|bonus","c|score","c|bonus"]
console.log([...s.values]);        // [10, 2, 30, 3]

// keep nulls
const sFull = stack(df, { dropna: false });
console.log([...sFull.index.values]);
// ["a|score","a|bonus","b|score","b|bonus","c|score","c|bonus"]
console.log([...sFull.values]);
// [10, null, null, 2, 30, 3]
▶ run

3 — unstack: recover the original DataFrame

unstack(s) is the inverse of stack(df, { dropna: false }). It parses the compound index labels and reconstructs the grid.

import { DataFrame, stack, unstack } from "tsb";

const df = DataFrame.fromColumns(
  { A: [1, 2], B: [3, 4] },
  { index: ["x", "y"] },
);

const s = stack(df, { dropna: false });
const recovered = unstack(s);

console.log([...recovered.index.values]);   // ["x", "y"]
console.log([...recovered.columns.values]); // ["A", "B"]
console.log(recovered.toRecords());
// [{ A: 1, B: 3 }, { A: 2, B: 4 }]
▶ run

4 — unstack fills missing cells

When stack was called with dropna=true (the default), some (row, col) combinations are absent. unstack fills them with null by default; pass fill_value to use a different filler.

import { DataFrame, stack, unstack } from "tsb";

const df = DataFrame.fromColumns(
  { revenue: [100, null, 300], cost: [null, 50, 60] },
  { index: ["jan", "feb", "mar"] },
);

// stack drops nulls by default
const s = stack(df);

// unstack with fill_value=0
const recovered = unstack(s, { fill_value: 0 });
console.log(recovered.toRecords());
// [{ revenue:100, cost:0 }, { revenue:0, cost:50 }, { revenue:300, cost:60 }]
▶ run

5 — Custom separator

If your row-index labels or column names contain "|", choose a different separator for both stack and unstack.

import { DataFrame, stack, unstack } from "tsb";

const df = DataFrame.fromColumns(
  { "col-A": [10, 20], "col-B": [30, 40] },
  { index: ["row-1", "row-2"] },
);

const s = stack(df, { sep: "::", dropna: false });
console.log([...s.index.values]);
// ["row-1::col-A","row-1::col-B","row-2::col-A","row-2::col-B"]

const recovered = unstack(s, { sep: "::" });
console.log([...recovered.index.values]);   // ["row-1","row-2"]
console.log([...recovered.columns.values]); // ["col-A","col-B"]

6 — Reshape workflow: stack → filter → unstack

A common pattern: stack to long format, filter rows, then unstack back to wide.

import { DataFrame, stack, unstack } from "tsb";

// Quarterly sales by product
const df = DataFrame.fromColumns(
  {
    widgets:  [120, 80,  150, 200],
    gadgets:  [60,  null, 90,  110],
    gizmos:   [30,  45,  null, 70],
  },
  { index: ["Q1", "Q2", "Q3", "Q4"] },
);

// Stack (drops nulls), filter to values > 80, then unstack
const s = stack(df);  // dropna=true by default
const filtered = s.filter(s.values.map((v) => (v as number) > 80));
const result = unstack(filtered, { fill_value: 0 });

console.log([...result.index.values]);
console.log(result.toRecords());
