Window Apply

Apply custom functions over rolling and expanding windows: rollingApply, expandingApply, dataFrameRollingApply, dataFrameExpandingApply.

Rolling apply — custom aggregation

rollingApply(series, window, fn) applies fn to each trailing window of non-null values. Positions with fewer than minPeriods valid values produce null.
