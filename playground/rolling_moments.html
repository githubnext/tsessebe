📊 Rolling Skew & Kurtosis

rollingSkew and rollingKurtosis compute sliding-window Fisher-Pearson skewness and bias-corrected excess kurtosis, mirroring pandas.Series.rolling(n).skew() and .kurt().

What Are These?

Skewness measures the asymmetry of a distribution: positive skew means a longer right tail, negative skew means a longer left tail. Requires ≥ 3 values per window.

Kurtosis (excess) measures the "peakedness" vs. a normal distribution: positive kurtosis = heavier tails (leptokurtic), negative = lighter tails (platykurtic). Requires ≥ 4 values per window.

Rolling Skewness — rollingSkew(series, window)

// Mirrors: pd.Series.rolling(window).skew()
import { Series, rollingSkew } from "tsb";

const prices = new Series({ data: [100, 102, 98, 105, 103, 110, 95, 108] });

const skew3 = rollingSkew(prices, 3);
// window=3: first 2 positions → null (need ≥ 3)
// position 2: [100, 102, 98] → slightly left-skewed
// [null, null, -0.707, 0.935, -0.935, 0.935, -0.935, 0.688]

// Custom minPeriods (default: 3)
const skew5mp4 = rollingSkew(prices, 5, { minPeriods: 4 });
// position 3 has 4 values → can compute

// Centred window
const skewCentered = rollingSkew(prices, 3, { center: true });
// window spans i-1, i, i+1 instead of i-2, i-1, i

Rolling Kurtosis — rollingKurtosis(series, window)

// Mirrors: pd.Series.rolling(window).kurt()
import { Series, rollingKurtosis } from "tsb";

// Uniform spacing [1,2,3,4,5] → excess kurtosis ≈ -1.2 (platykurtic)
const s = new Series({ data: [1, 2, 3, 4, 5] });

const kurt5 = rollingKurtosis(s, 5);
// [null, null, null, null, -1.2]

// Larger window on financial returns
const returns = new Series({ data: [0.01, -0.02, 0.05, -0.03, 0.08, -0.01, 0.02] });
const kurt4 = rollingKurtosis(returns, 4);
// Detects heavy-tailed or light-tailed windows

DataFrame Support

import { DataFrame, rollingSkewDataFrame, rollingKurtosisDataFrame } from "tsb";

const df = DataFrame.fromColumns({
  tech:    [150, 148, 155, 160, 145, 170, 140, 165],
  finance: [80,  82,  79,  85,  81,  90,  75,  88],
});

// Column-wise rolling skew (axis=0, default)
const skewDf = rollingSkewDataFrame(df, 4);
// tech and finance columns each get rolling skew applied

// Column-wise rolling kurtosis
const kurtDf = rollingKurtosisDataFrame(df, 5);

Formulas

Fisher-Pearson Adjusted Skew (matches pandas .skew()):

g1 = m3 / σ³   (biased standardised 3rd moment)
G1 = √(n(n-1)) / (n-2) × g1

Bias-Corrected Excess Kurtosis (matches pandas .kurt()):

G2 = (n+1)·n·(n-1) / ((n-2)·(n-3)) × (m4/m2²) − 3(n-1)² / ((n-2)·(n-3))
