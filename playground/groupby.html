🔗 GroupBy — Split-Apply-Combine

GroupBy is pandas' answer to SQL's GROUP BY clause. It implements the split → apply → combine pattern: split a data structure into groups, apply a function to each group independently, then combine the results.

Quick Start

import { DataFrame } from "tsb";
import { dataFrameGroupBy } from "tsb";

const df = DataFrame.fromRecords([
  { city: "NYC",   sales: 100 },
  { city: "LA",    sales:  80 },
  { city: "NYC",   sales:  60 },
  { city: "LA",    sales: 120 },
  { city: "Chicago", sales: 50 },
]);

const grp = dataFrameGroupBy(df, "city");

grp.sum().toString();
// city     │ sales
// ─────────┼───────
// NYC      │ 160
// LA       │ 200
// Chicago  │ 50

Creating a GroupBy Object

For DataFrames

// Group by a single column
const g1 = dataFrameGroupBy(df, "city");

// Group by multiple columns (composite key)
const g2 = dataFrameGroupBy(df, ["region", "city"]);

For Series

import { Series } from "tsb";
import { seriesGroupBy } from "tsb";

const s = new Series({ data: [10, 20, 30, 40], name: "sales" });

// Group by an external array of labels
const g1 = seriesGroupBy(s, ["East", "West", "East", "West"]);

// Group by the series' own index
const indexed = new Series({ data: [1, 2, 3], index: ["a", "b", "a"] });
const g2 = seriesGroupBy(indexed, "index");

// Group by another Series
const keys = new Series({ data: ["Q1", "Q2", "Q1", "Q2"] });
const g3 = seriesGroupBy(s, keys);

Built-in Aggregations

const grp = dataFrameGroupBy(df, "city");

grp.sum()    // total per group
grp.mean()   // arithmetic mean per group
grp.min()    // minimum per group
grp.max()    // maximum per group
grp.count()  // non-null count per group
grp.first()  // first value per group
grp.last()   // last value per group
grp.std()    // sample standard deviation per group
city │ sales ─────────┼─────── NYC │ 160 LA │ 200 Chicago │ 50

Custom Aggregation with agg()

// Same function for all columns
grp.agg("sum")
grp.agg(vals => vals.length)   // custom: count rows per group

// Different functions per column
grp.agg({ sales: "sum", discount: "mean" })

Transform — Same Shape as Input

Unlike aggregation (which returns one row per group), transform() returns a result with the same number of rows as the source. Each element is replaced by the result of applying the function to its group. 

const scores = new Series({ data: [80, 90, 70, 85, 95] });
const teams  = ["A", "B", "A", "B", "A"];

// Subtract the group mean (demeaning / z-score numerator)
const demeaned = seriesGroupBy(scores, teams).transform(vals => {
  const mean = vals.reduce((a, b) => (a as number) + (b as number), 0) / vals.length;
  return vals.map(v => (v as number) - (mean as number));
});
// [80-81.67, 90-87.5, 70-81.67, 85-87.5, 95-81.67]
// ≈ [-1.67, 2.5, -11.67, -2.5, 13.33]

Apply — Arbitrary Function per Group

// Compute a custom scalar per group
const result = seriesGroupBy(scores, teams).apply(group => {
  const max = Math.max(...group.values as number[]);
  const min = Math.min(...group.values as number[]);
  return max - min;
});
// A: range = 95 - 70 = 25
// B: range = 90 - 85 = 5

Inspecting Groups

const grp = dataFrameGroupBy(df, "city");

grp.ngroups          // 3
grp.groupKeys        // ["NYC", "LA", "Chicago"]
grp.groups           // Map: "NYC" → [0, 2], "LA" → [1, 3], …

// Get the sub-DataFrame for one group
grp.getGroup("NYC")

// Iterate over all groups
for (const [key, subDf] of grp) {
  console.log(key, subDf.shape);
}

API Reference

Function / MethodDescription
seriesGroupBy(s, by)Create a SeriesGroupBy from a Series and keys
dataFrameGroupBy(df, by)Create a DataFrameGroupBy from a DataFrame and column(s)
.sum()Sum per group
.mean()Mean per group
.min() / .max()Min / max per group
.count()Non-null count per group
.first() / .last()First / last value per group
.std()Sample standard deviation per group
.agg(func | name | spec)Aggregate with custom function(s)
.transform(func)Transform, preserving shape
.apply(func)Apply arbitrary function per group
.getGroup(key)Retrieve a single group
[Symbol.iterator]Iterate over [key, group] pairs
