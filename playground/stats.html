📊 Statistical Functions

tsb implements the core statistical functions from pandas: describe, corr, cov, skew, and kurtosis.

describe — Summary Statistics

describe(series) computes summary statistics for a Series. Numeric Series returns count, mean, std, min, 25%, 50%, 75%, max. Non-numeric Series returns count, unique, top, freq. 

import { Series, DataFrame, describe, describeDataFrame } from "tsb";

const scores = new Series({ data: [72, 85, 90, 68, 95, 78, 88], name: "scores" });
const stats = describe(scores);
// Series {
//   count: 7,   mean: 82.28...,  std: 9.46...,
//   min: 68,   "25%": 75,       "50%": 85,    "75%": 89,  max: 95
// }

const df = DataFrame.fromColumns({
  math:    [90, 80, 70, 85],
  english: [75, 88, 92, 60],
  science: [82, 79, 95, 88],
});
const summary = describeDataFrame(df);
// DataFrame (8 × 3) — rows are stat names, columns are math/english/science

Output shape

StatisticDescription
countNumber of non-missing values
meanArithmetic mean
stdSample standard deviation (ddof=1)
min / maxMinimum / maximum value
25% / 50% / 75%Percentiles (linear interpolation)

corr — Correlation

corrSeries(a, b, method?) computes the pairwise correlation coefficient. corrDataFrame(df, method?) returns a correlation matrix. Three methods are supported:

  • pearson (default) — linear correlation
  • spearman — rank-based, monotone relationships
  • kendall — Kendall's tau-b, ordinal association
import { Series, DataFrame, corrSeries, corrDataFrame } from "tsb";

const temp   = new Series({ data: [20, 25, 30, 35, 40] });  // temperature °C
const iceCream = new Series({ data: [100, 150, 200, 250, 300] });  // units sold

corrSeries(temp, iceCream);               // 1.0 — perfect linear correlation
corrSeries(temp, iceCream, "spearman");   // 1.0 — perfect rank correlation
corrSeries(temp, iceCream, "kendall");    // 1.0 — perfect concordance

// Correlation matrix
const df = DataFrame.fromColumns({
  x: [1, 2, 3, 4, 5],
  y: [5, 4, 3, 2, 1],   // perfectly anti-correlated with x
  z: [2, 2, 2, 2, 2],   // constant — undefined correlation (NaN)
});
corrDataFrame(df);
// DataFrame 3×3:
//     x     y     z
// x   1    -1    NaN
// y  -1     1    NaN
// z  NaN  NaN     1

cov — Covariance

covSeries(a, b, ddof?) computes sample covariance (ddof=1 by default). covDataFrame(df, ddof?) returns a covariance matrix.

The diagonal of covDataFrame equals the column-wise Series.var(). 

import { Series, DataFrame, covSeries, covDataFrame } from "tsb";

const a = new Series({ data: [1, 2, 3, 4, 5] });
const b = new Series({ data: [5, 4, 3, 2, 1] });

covSeries(a, b);      // -2.5 (negative covariance — inversely related)
covSeries(a, a);      // 2.5  (== a.var())
covSeries(a, b, 0);   // -2.0 (population covariance)

const df = DataFrame.fromColumns({ a: [1,2,3,4,5], b: [5,4,3,2,1] });
covDataFrame(df);
// DataFrame 2×2:
//     a     b
// a   2.5  -2.5
// b  -2.5   2.5

skew — Skewness

skewSeries(s) computes the adjusted Fisher–Pearson skewness coefficient (same formula as pandas.Series.skew with bias=False). Requires at least 3 non-missing values; returns NaN otherwise.

  • > 0 — right-skewed (long right tail)
  • = 0 — symmetric
  • < 0 — left-skewed (long left tail)
import { Series, DataFrame, skewSeries, skewDataFrame } from "tsb";

const symmetric    = new Series({ data: [1, 2, 3, 4, 5] });
const rightSkewed  = new Series({ data: [1, 1, 1, 1, 10] });
const leftSkewed   = new Series({ data: [-10, 1, 1, 1, 1] });

skewSeries(symmetric);    // ≈ 0
skewSeries(rightSkewed);  // > 0
skewSeries(leftSkewed);   // < 0

// Per-column skewness
const df = DataFrame.fromColumns({
  normal: [1, 2, 3, 4, 5],
  skewed: [1, 1, 1, 2, 10],
});
skewDataFrame(df);
// Series { normal: ≈0, skewed: >0 } indexed by column name

kurtosis / kurt — Excess Kurtosis

kurtosisSeries(s) (alias kurtSeries) computes adjusted excess kurtosis (Fisher definition, same as pandas.Series.kurt). Requires at least 4 non-missing values.

  • > 0 — leptokurtic (heavy tails, sharp peak)
  • = 0 — mesokurtic (normal-like)
  • < 0 — platykurtic (light tails, flat peak)
import { Series, DataFrame, kurtosisSeries, kurtSeries, kurtosisDataFrame, kurtDataFrame } from "tsb";

const uniform = new Series({ data: [1, 2, 3, 4, 5] });        // platykurtic
const peaky   = new Series({ data: [0, 0, 0, 0, 0, 0, 100] }); // leptokurtic

kurtosisSeries(uniform);  // ≈ -1.3 (platykurtic)
kurtSeries(peaky);        // > 0 (leptokurtic)

// Per-column excess kurtosis
const df = DataFrame.fromColumns({
  a: [1, 2, 3, 4, 5],
  b: [0, 0, 0, 0, 100],
});
kurtosisDataFrame(df);   // Series { a: ≈-1.3, b: >0 }
kurtDataFrame(df);       // same — alias

Handling Missing Values

All functions automatically skip null, undefined, and NaN values. For corrSeries / covSeries, pairwise-complete observations are used — a row is only excluded if either Series has a missing value at that position. 

import { Series, corrSeries } from "tsb";

const a = new Series({ data: [1, null, 3, 4, 5] });
const b = new Series({ data: [2, 4,    6, 8, 10] });

corrSeries(a, b);
// Uses positions 0, 2, 3, 4 (position 1 skipped because a[1] is null)
// Result: 1.0 (still perfect linear correlation on remaining values)
Formula note: Skewness and kurtosis use the adjusted Fisher–Pearson standardized moment coefficients (unbiased for small samples), matching pandas' default bias=False behaviour.
