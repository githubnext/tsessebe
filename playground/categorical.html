🏷️ Categorical Data

tsb implements pandas' Categorical data type: a finite-vocabulary array that stores values as integer codes into an ordered list of categories. Categories save memory when the same strings repeat many times and enable ordered comparisons like "low < medium < high".

Creating a Categorical

Pass any array of values to new Categorical(values). Categories are inferred in order of first appearance. Missing values (null / undefined) are assigned code -1. 

import { Categorical } from "tsb";

const c = new Categorical(["a", "b", "a", "c", null]);
c.categories;  // ["a", "b", "c"]
c.codes;       // [0, 1, 0, 2, -1]
c.length;      // 5
c.nCategories; // 3
c.at(0);       // "a"
c.at(4);       // null  (missing)
c.toArray();   // ["a", "b", "a", "c", null]

Explicit Categories & Ordering

Provide a categories option to fix the category list (any value not in the list becomes -1). Set ordered: true to enable min() / max() operations. 

const priority = new Categorical(
  ["medium", "high", "low", "high"],
  { categories: ["low", "medium", "high"], ordered: true }
);
priority.ordered; // true
priority.min();   // "low"
priority.max();   // "high"

// Values outside the explicit categories become missing:
const c = new Categorical(["a", "b", "z"], { categories: ["a", "b"] });
c.codes;    // [0, 1, -1]
c.toArray(); // ["a", "b", null]

Mutating the Category Set

All mutation methods return a new Categorical — the original is never changed. 

const c = new Categorical(["a", "b", "a"], { categories: ["a", "b"] });

// Add new categories (data codes unchanged)
c.addCategories(["c", "d"]).categories;  // ["a", "b", "c", "d"]

// Remove a category (matching values → -1 / null)
c.removeCategories(["b"]).toArray();  // ["a", null, "a"]

// Remove categories with zero occurrences
const sparse = new Categorical(["a"], { categories: ["a", "b", "c"] });
sparse.removeUnusedCategories().categories;  // ["a"]

// Rename categories (one-to-one, must keep same length)
c.renameCategories(["x", "y"]).toArray();  // ["x", "y", "x"]

// Reorder (must be a permutation of current categories)
c.reorderCategories(["b", "a"]).codes;  // [1, 0, 1]

// Replace the entire category set
c.setCategories(["a"]).toArray();  // ["a", null, "a"]

Statistics

const c = new Categorical(["a", "a", "b", "b", "b", null]);

// Count per category (includes zero-count categories)
c.valueCounts();
// Series { a: 2, b: 3 }

// Summary stats (like pandas describe for categorical)
c.describe();
// Series { count: 5, unique: 2, top: "b", freq: 3 }

Series.cat Accessor

The .cat accessor on any Series exposes categorical operations — mirroring pandas.Series.cat. 

import { Series } from "tsb";

const s = new Series({ data: ["a", "b", "a", "c"], name: "grade" });

s.cat.categories;       // ["a", "b", "c"]
s.cat.ordered;          // false
s.cat.codes.values;     // [0, 1, 0, 2]

s.cat.valueCounts();    // Series { a: 2, b: 1, c: 1 }

// Operations return new Series:
const renamed = s.cat.renameCategories(["x", "y", "z"]);
renamed.values;  // ["x", "y", "x", "z"]

const ordered = s.cat.asOrdered();
ordered.cat.ordered;  // true

factorize

factorize(values) encodes any array (or Series) into integer codes plus a list of unique labels — the building block for many label-encoding workflows. 

import { factorize } from "tsb";

const { codes, uniques } = factorize(["cat", "dog", "cat", null, "bird"]);
codes;   // [0, 1, 0, -1, 2]
uniques; // ["cat", "dog", "bird"]

// Works with Series too:
const s = new Series({ data: [10, 20, 10, 30] });
const { codes: c2, uniques: u2 } = factorize(s);
c2;  // [0, 1, 0, 2]
u2;  // [10, 20, 30]

CategoricalDtype

CategoricalDtype is a metadata descriptor that can accompany a Series or DataFrame column to record what categories and ordering it expects. 

import { CategoricalDtype, Dtype } from "tsb";

const dt = new CategoricalDtype(["low", "medium", "high"], true);
dt.categories; // ["low", "medium", "high"]
dt.ordered;    // true
dt.dtype;      // Dtype.category
dt.toString(); // 'CategoricalDtype(categories=["low","medium","high"], ordered=True)'

// Compare two CategoricalDtype instances:
const dt2 = new CategoricalDtype(["low", "medium", "high"], true);
dt.equals(dt2); // true
