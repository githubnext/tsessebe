Sorting Utilities New

tsb provides high-level sorting utilities that complement the built-in sortValues/sortIndex methods on Series and DataFrame. These match the pandas API for nlargest, nsmallest, and rank.

1. nlargest / nsmallest (Series)

Select the top-n or bottom-n values from a Series, preserving the original index order of the selected rows.

2. nlargest with keep options

When there are ties at the boundary, control which tied value is selected with the keep option: "first", "last", or "all".

3. dataFrameNlargest / dataFrameNsmallest

Select the n rows with the largest (or smallest) values in one or more columns, returning a full DataFrame.

4. rank — Series ranking

Compute numeric ranks for each element. Ties are broken according to method. Ranks are 1-based; use pct: true for fractional (0, 1] ranks.

5. rank with NaN handling

Control where missing values appear in the ranked output using naOption: "keep" (default), "top", or "bottom".

6. rankDataFrame

Rank each column of a DataFrame independently, returning a new DataFrame of float ranks.

API Reference

Function Description pandas equivalent
nlargest(series, n, {keep?}) n largest values from a Series Series.nlargest()
nsmallest(series, n, {keep?}) n smallest values from a Series Series.nsmallest()
dataFrameNlargest(df, n, columns, {keep?}) n rows with largest column values DataFrame.nlargest()
dataFrameNsmallest(df, n, columns, {keep?}) n rows with smallest column values DataFrame.nsmallest()
rank(series, {method?, ascending?, naOption?, pct?}) Rank each element of a Series Series.rank()
rankDataFrame(df, {method?, ascending?, naOption?, pct?}) Rank each column of a DataFrame DataFrame.rank()
