Reshaping New

tsb provides four reshaping utilities that mirror the pandas reshaping API: pivot, pivotTable, melt, stack, and unstack. These convert DataFrames between wide and long formats.

1. pivot — wide from long

pivot(df, {index, columns, values}) reshapes a long-format DataFrame into a wide-format one. Each unique value in the columns column becomes a new column header. Raises an error if there are duplicate (index, columns) pairs — use pivotTable for aggregation.

2. pivotTable — aggregate duplicates

pivotTable(df, {values, index, columns, aggfunc}) works like pivot but aggregates duplicate (index, columns) combinations. Supported aggregation functions: "mean" (default), "sum", "count", "min", "max", "first", "last".

3. melt — wide to long

melt(df, {id_vars, value_vars, var_name, value_name}) unpivots a wide DataFrame into a long format. Identifier columns are preserved; value columns are "melted" into two new columns holding the original column name and value.

4. stack — collapse column axis into rows

stack(df, options) converts a wide DataFrame to a long format, adding a "_index" column with original row labels, a "variable" column with column names, and a "value" column with values. Because tsb does not yet support MultiIndex, the original row-index is stored as a separate column rather than becoming a second index level (as in pandas).

5. unstack — wide again

unstack(df, options) is the inverse of stack. It reads the "_index", "variable", and "value" columns and reconstructs the original wide DataFrame.

API Reference

Function Description pandas equivalent
pivot(df, {index, columns, values}) Reshape long → wide; raises on duplicates DataFrame.pivot()
pivotTable(df, {values, index, columns, aggfunc?, fillValue?}) Reshape long → wide with aggregation pandas.pivot_table()
melt(df, {id_vars?, value_vars?, var_name?, value_name?}) Unpivot wide → long format DataFrame.melt()
stack(df, {value_vars?, var_name?, value_name?, index_name?}) Collapse column axis into rows DataFrame.stack()
unstack(df, {index_col?, var_col?, value_col?, fill_value?}) Expand row column back into columns DataFrame.unstack()
