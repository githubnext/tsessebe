cumulative operations

Compute running totals, products, maxima, and minima — mirrors pandas.Series.cumsum() / cumprod() / cummax() / cummin().

1 — cumsum: running total

cumsum(series) returns a new Series where each value is the sum of all preceding values plus the current one. Mirrors pandas.Series.cumsum(). 

import { Series, cumsum } from "tsb";

const s = new Series({ data: [1, 2, 3, 4, 5] });
const cs = cumsum(s);
console.log([...cs.values]); // [1, 3, 6, 10, 15]

2 — cumprod: running product

cumprod(series) returns a new Series where each value is the product of all values up to and including that position. Mirrors pandas.Series.cumprod(). 

import { Series, cumprod } from "tsb";

const s = new Series({ data: [1, 2, 3, 4, 5] });
const cp = cumprod(s);
console.log([...cp.values]); // [1, 2, 6, 24, 120]

3 — cummax and cummin

cummax(series) tracks the running maximum; cummin(series) tracks the running minimum. Both work on numbers, strings, and booleans. 

import { Series, cummax, cummin } from "tsb";

const s = new Series({ data: [3, 1, 4, 1, 5, 9, 2, 6] });
console.log([...cummax(s).values]); // [3, 3, 4, 4, 5, 9, 9, 9]
console.log([...cummin(s).values]); // [3, 1, 1, 1, 1, 1, 1, 1]

4 — Handling missing values (skipna)

By default skipna: true: missing values return NaN/null in the result but do not affect the running accumulator. With skipna: false, any missing value poisons all subsequent results. 

import { Series, cumsum } from "tsb";

const s = new Series({ data: [1, null, 3, 4] });

// skipna: true (default) — NaN at position 1, accumulator continues
const skipTrue = cumsum(s);
console.log([...skipTrue.values]); // [1, NaN, 4, 8]

// skipna: false — NaN at position 1 poisons everything after
const skipFalse = cumsum(s, { skipna: false });
console.log([...skipFalse.values]); // [1, NaN, NaN, NaN]

5 — DataFrame: axis=0 (column-wise)

dataFrameCumsum(df) applies the operation independently to each column (axis=0 is the default, same as pandas). 

import { DataFrame, dataFrameCumsum, dataFrameCummax } from "tsb";

const df = DataFrame.fromColumns({
  revenue: [100, 150, 200, 120],
  cost:    [60,  80,  110, 70],
});

const csDf = dataFrameCumsum(df);
console.log([...csDf.col("revenue").values]); // [100, 250, 450, 570]
console.log([...csDf.col("cost").values]);    // [60, 140, 250, 320]

const cmDf = dataFrameCummax(df);
console.log([...cmDf.col("revenue").values]); // [100, 150, 200, 200]

6 — DataFrame: axis=1 (row-wise)

With axis: 1 (or axis: "columns"), the operation is applied across columns for each row — each cell becomes the cumulative value of all columns to its left plus itself. 

import { DataFrame, dataFrameCumsum } from "tsb";

const df = DataFrame.fromColumns({
  q1: [10, 20, 30],
  q2: [15, 25, 35],
  q3: [12, 22, 32],
});

// axis=1: running total across quarters for each year-row
const ytd = dataFrameCumsum(df, { axis: 1 });
console.log([...ytd.col("q1").values]); // [10, 20, 30]   (unchanged — first col)
console.log([...ytd.col("q2").values]); // [25, 45, 65]   (q1+q2)
console.log([...ytd.col("q3").values]); // [37, 67, 97]   (q1+q2+q3)

7 — Real-world example: portfolio tracking

Track the running portfolio value and the running drawdown (how far we are from the all-time high) using cumsum and cummax. 

import { Series, cumsum, cummax } from "tsb";

// Daily P&L in dollars
const pnl = new Series({ data: [200, -50, 300, -150, 400, -80, 600] });

const equity = cumsum(pnl);
const peak   = cummax(equity);

// Drawdown: how far below the all-time high we currently are
const drawdown = new Series({
  data: equity.values.map((v, i) => {
    const p = peak.values[i];
    return (typeof v === "number" && typeof p === "number") ? v - p : null;
  }),
});

console.log("Equity curve: ", [...equity.values]);
// [200, 150, 450, 300, 700, 620, 1220]
console.log("All-time high:", [...peak.values]);
// [200, 200, 450, 450, 700, 700, 1220]
console.log("Drawdown:     ", [...drawdown.values]);
// [0, -50, 0, -150, 0, -80, 0]

8 — String series: lexicographic cummax / cummin

cummax and cummin work on any comparable type, including strings (lexicographic ordering). 

import { Series, cummax, cummin } from "tsb";

const words = new Series({ data: ["banana", "apple", "cherry", "apricot", "date"] });
console.log([...cummax(words).values]);
// ["banana", "banana", "cherry", "cherry", "cherry"]
console.log([...cummin(words).values]);
// ["banana", "apple",  "apple",  "apple",  "apple"]
