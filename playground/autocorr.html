🔄 autocorr

autocorr(series, lag) computes the Pearson autocorrelation coefficient of a series at the given lag offset. Mirrors pandas.Series.autocorr(lag). 

// autocorr(s, k) = pearsonr(s[k:], s[:-k])
// Missing values are silently dropped before computing.

import { Series, autocorr } from "tsb";

const prices = new Series([100, 102, 101, 105, 107, 110, 108, 112]);

console.log(autocorr(prices, 1));   // ~0.95 (strong positive autocorrelation)
console.log(autocorr(prices, 2));   // ~0.89
console.log(autocorr(prices, 0));   // 1    (always 1 at lag 0)

Edge Cases

ScenarioResultReason
lag = 01A series is perfectly correlated with itself.
|lag| ≥ lengthNaNNo overlapping pairs exist.
Constant series, lag ≥ 1NaNZero variance — Pearson undefined.
All NaN / nullNaNNo valid pairs remain after skipping missings.
negative lagSame as positivePearson correlation is symmetric.

API Reference

autocorr(series: Series<Scalar>, lag?: number, options?: AutocorrOptions): number

interface AutocorrOptions {
  minPeriods?: number;  // default: 1
}

Note: Only numeric values participate in the computation. Non-numeric scalars (strings, booleans) are treated as missing.
